목록 출력

  • ul(unordered list): type속성(disc, circle, square)
  • ol(ordered list): type속성(1,a,A,i,I)
  • dl(definition list): 용어정리를의미, dt,dd를 자식으로 가져야 함.

ul 순서가 없는 목록(순서가 중요하지 않은 목록)

예시)

올해 개봉작

  • 플래시
  • 가디언즈오브..
  • 범죄도시

ol 순서가 있는 목록(순서가 중요한 목록)

예시)

올해 영화 흥행 순위

  1. 가디언즈오브
  2. 플래시
  3. 범죄도시

dl (용어의 의미)


예시)

html
html이란...(설명을 적는다.)
css
css이란...(설명을 적는다.)
