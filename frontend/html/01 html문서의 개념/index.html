Hyper Text Markup Language

html문서의 특징과 주의사항

  • 시멘틱언어, 웹표준, 웹접근성, 크로스브라우징을 준수해야 함
  • DOM요소 : element의 집합
  • element : tagname, 속성(attribute)
  • DOM은 정해진 node(parent [부모], children [자식], sibling [형제], ancestor [조상], decesdent [손자])를 갖는다

    • 다양한 시멘틱 요소

    • h1-h6(heading): h1요소는 웹문서에 한개를 포함하길 권장
    • p(paragraph): 두개 이상의 단어가 모인 하나의 문장
    • ul,li : 순서가 없는 목록
    • ol,li : 순서가 있는 목록
    • main, section, article : 구획요소
    • em, b, strong : 강조, 중요
    • small : 보조적인 내용
    • i: 구분
    • address : 주소, 이름
    • time : 시간
    • blockquote, q : 인용문
    • cite : 출처
    • code : 프로그램 언어
    • kbd : 키보드
    • sub, sup : 윗첨자, 아랫첨자
    • div, span : 의미가 없음
