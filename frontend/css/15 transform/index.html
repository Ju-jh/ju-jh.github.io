transform

  • 레이아웃을 유지하면서 모양이 변형된 형태로 렌더링
  • transform: scale(1.2) rotate(45deg) translate(0,0)
  • transform속성 값의 %단위 기준요소는 해당요소
  • transform은 inline 레벨 요소에는 적용이 안된다.
  • transform속성을 적용하면 z-index가 적용된다.
