float

  • float이 적용된 객체의 부모는 높이가 사라진다
    clearfix를 이용해서 부모의 높이 문제를 해결해야함
  • float이 적용된 객체는 display속성이 적용되지 않는다.
    (block 레벨의 넓이가 없어짐, inline레벨도 크기를 가질 수 있다. margin의 auto값이 적용되지 않는다.)
  • float이 적용되지 않은 용소는, float요소와 margin이 적용되지 않는다.
