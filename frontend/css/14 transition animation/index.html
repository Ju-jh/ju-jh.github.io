transition

  • 숫자값으로 설정된 css속성의 전환효과
  • transition:속성1 초s, 속성2 초s,
    transition:all 초s
  • transition:속성1 초s, 속성2 초s,
    transition-delay:0s, 1s;
  • transition-timing-function: 감가속
    ease-out (감속)
    ease-in (가속)
    linear (등속)
  • 단점 :
    animation 중단시 트랜지션이(전환효과) 적용되지 않는다.

animation

  • 복잡한 모션효과 구현
  • @keyframe을 이용해서 애니메이션 설계
    animation 속성을 이용해서 애니메이션 호출
  • animation-name: 키프레임 이름
    animation-duration: 애니메이션 시간
    animation-iteration-count: 반복 횟수
    animation-timing-function: 감가속;
    animation-direction: 키프레임 진행 방향;
