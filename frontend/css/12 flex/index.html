flex

  • flex요소는 block요소와 유사하다
  • flex의 자식요소는 기존의 display속성을 상실한다

flex direction

  • 컨테이너 안의 그리드의 배치 방향
  • row, row-reverse, column, column-reverse
  • 기본값은 row (flex-direction : row)

align-items

  • 축의 위치
  • 기본값 stretch
    flex-direction이 row일 경우 : 부모의 높이와 같아진다
    flex-direction이 column일 경우 : 부모의 넓이와 같아진다
  • flex-direction이 row일 경우
    • flex-start : 상단축
    • flex-end : 하단축
    • center : 수직 중앙축
  • flex-direction이 column일 경우
    • flex-start : 왼쪽축
    • flex-end : 오른쪽축
    • center : 수평중앙축

justify-content

  • 축의방향으로 grid의 위치를 결정
  • flex-direction이 row일 경우
    • flex-start : 왼쪽
    • flex-end : 오른쪽
    • center : 중앙
    • space-btween : 위아래로 분할
  • flex-direction이 column일 경우
    • flex-start : 상단
    • flex-end : 하단
    • center : 중앙
    • space-btween : 상하로 분할
