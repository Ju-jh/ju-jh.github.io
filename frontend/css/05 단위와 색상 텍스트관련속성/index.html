unit, color, text properties

글자관련속성

  • color: 글자색
  • font-size: 10px
  • font-family: 글꼴
  • letter-spacing: 자간
  • word-family: 어간
  • text-align: 정렬
  • line-height: 행간
  • font-weight: 글자두께
  • font-style: italic
  • text-decoration: 줄

unit

  • pt : 72분의 1인치(크로스 브라우징 안됨, 비추)
  • px : 논리전 단위의 최소 단위
  • em : 부모크기의 배율
  • rem : 16px의 배율

색상표기법

  • 고유색상명

    black, red등의 색상명

  • rgb표기법

    8bit 칼라 :Red:0~255, Blue:0~255, Green:0~255
    rgb예시칼라 : rgb(255,124,255), rgba(255,0,0.5). rgb(128,128,128)

  • 색상번호표기법

    16진수를 이용한 표기법 : #ffffff : #RRGGBB
    웹표기법: #ff3366 = #F36
