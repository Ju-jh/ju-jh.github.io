css기초

Cascading Style Sheet

특징과 개념

  • html문서를 기반으로 한 디자인 언어
  • 우선순위가 존재한다
  • 상속(inheritance)개념이 존재한다
  • 다양한 선택자(selector)가 존재한다

선택자

태그선택자

  • 문법

    style엘리먼트 안쪽에서 tagname{속성:속성값}

  • 우선순위 : 자식선택자, 부모선택자
  • 모든 태그에 동일한 스타일이 적용된다

class selector

  • 문법 : style요소안에서 .클래스네임{속성:속성값} 선언 후 html요소의 class속성으로 호출
  • 반복해서 호출이 가능함(수정, 관리가 용이함)
  • 우선순위 : 클래스, 태그선택자

아이디 선택자

  • style요소내에서 #아이디{속성:속성값}선언 후, html id속성으로 호출
  • id는 중복호출 될 수 없음
  • 우선순위 : 아이디, 클래스

inline style

  • 문법 : html요소의 style속성을 추가 후 css문법을 작성
  • script로 스타일을 제어할떄 사용
  • 선택자보다 우선순위가 높음

우선순위

0. !important
1. inline style
2. id selector
3. class selector
4. type selector
5. children selector
