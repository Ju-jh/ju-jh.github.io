grid layout


  • grid속성은 block레벨과 유사, 노드 순서와 상관없이 레이아웃을 구현할 수 있다.
  • 그리드분할예시 : repeat(3,auto),repeat(3,1fr),repeat(3,100px)
  • grid요소의 자식은 크기를 지정하지 않을 시 최소그리드 한개의 크기를 갖는다.
  • 자식요소의 grid칸설정 예시
    grid-colum : 2 / 4 ( 2~3 칸을 소유 ),
    grid-colum : 3 (3칸에서 시작하는 1칸짜리)
