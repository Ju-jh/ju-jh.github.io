margin과 padding

  • 첫번째, 마지막 블럭레벨요소는 상하마진이 적용되지 않는다 (예외 : 부모객체의 padding-top, border-top이 적용되지 않았을경우)
  • padding과 border는 기본적으로 크기에 추가됨.
    box-sizing:border-box를 이용해서 크기 안쪽에 포함시킬 수 있음.
  • inline 레벨은 상하마진이나 상하패딩이 적용되지 않는다.
  • block레벨간에 상하마진은 하나의 큰 값만 적용된다.
