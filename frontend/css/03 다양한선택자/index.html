다양한선택자

전체선택자(universial selector)

* : 모든 엘리먼트를 의미함
div * : div안쪽의 모든 요소
:hover : *:hover

combine selector

div.box : box라는 클래스명을 가진 div
div#box.box : 아이디가 box이면서 클래스는 box인 div

sibling selector

div + : div의 동생 (Adjacent sibling)

div ~ * : div의 동생들 (sibling)

div ~ P : div의 동생들중에 p요소 (Adjacent sibling)

first 가상클래스

div:first-child : 모든요소 순서를 기준으로 첫번째인 div
div:firts-of-type : div순서를 기준으로 첫번째인 div

last 가상클래스

div:last-child : 모든요소 순서를 기준으로 마지막번째인 div
div:last-of-type : div순서를 기준으로 마지막인 div

nth 가상클래스

div:nth-child(3) : 3번째 div
div:nth-of-type(1) : div중의 첫번째 div
div:nth-child(even) : 모든요소 순서를 기준으로 짝수번째
div:nth-of-type(even) : div중의 짝수번째 div
div:nth-child(odd) : 모든요소 순서를 기준으로 홀수번째
div:nth-of-type(odd) : div중의 홀수번째 div
div:nth-child(3n+2) : 모든요소 순서를 기준으로 3n+2째

not() 가상클래스

  • 특정요소는 제외하는 선택자
  • 예시) .ex4 b:nth-child(even){color:red}
    1 2 3 4 5 6 7 8
  • 주의: not 선택자 안에서 not을 사용할 수 없다

속성선택자

.ex5 a[href] : href속성을 가지고 있는 a요소

.ex5 a[href='http://'] : href속성의 값이 http://와 일치하는 a요소

.ex5 a[href^='http'] : href속성값이 http로 시작되는 a요소

.ex5 a[href$='http'] : href속성값이 http로 끝나는 a요소

.ex5 a[href*='http'] : href속성값에 http를 포함하는 a요소
